# Emotion-Aware Learning Companion

## Problem Statement
Students in India face immense academic pressure. Current learning tools 
ignore the emotional state of students, providing the same experience 
regardless of whether a student is stressed, confused, or confident.

## Solution
An AI-powered web app that detects student emotions from their text 
and adapts its teaching style accordingly using Google's Gemini 2.5 Flash API.

## Tech Stack
- Google Gemini 2.5 Flash API
- Node.js + Express
- HTML, CSS, JavaScript
- Deployed on Render

## Features
- Real-time emotion detection
- Adaptive teaching responses
- Support for 5 emotions
- Works for all subjects and levels
